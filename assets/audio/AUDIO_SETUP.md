# Audio Setup Instructions

## Required Audio File

To enable sound effects in the Dice Roller app, you need to add a dice rolling sound file.

### File Location
Place your audio file here: `assets/audio/dice_roll.mp3`

### Recommended Sources for Free Dice Roll Sounds

1. **FreeSound.org**
   - Visit: https://freesound.org/search/?q=dice+roll
   - Search for "dice roll" or "dice rolling"
   - Download as MP3 format

2. **Zapsplat**
   - Visit: https://www.zapsplat.com/sound-effect-category/dice/
   - Free dice rolling sound effects
   - Choose MP3 format

3. **Mixkit**
   - Visit: https://mixkit.co/free-sound-effects/dice/
   - Free sound effects library
   - Download and convert to MP3 if needed

### Audio File Specifications

- **Format**: MP3
- **Recommended Duration**: 1-3 seconds
- **File Name**: `dice_roll.mp3` (exactly)
- **Path**: `assets/audio/dice_roll.mp3`

### What Happens Without the Audio File?

The app will work perfectly fine without the audio file. If the file is not found:
- The app continues to function normally
- Sound effects will be silently skipped
- No error will be shown to users
- You can toggle sound on/off in settings regardless

### Adding the File Later

1. Download your preferred dice roll sound
2. Convert to MP3 format if necessary
3. Rename to `dice_roll.mp3`
4. Place in `assets/audio/` folder
5. Restart the app

The sound will automatically work once the file is in place!

### Testing Sound

After adding the file:
1. Open the app
2. Go to Settings
3. Ensure "Sound Effects" is toggled ON
4. Return to home screen
5. Tap the Play button to roll dice
6. You should hear the dice rolling sound

---

**Note**: Sound effects can always be disabled in the Settings screen, regardless of whether an audio file is present.
